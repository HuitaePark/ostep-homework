## 4. 타임 퀀텀의 크기 (-q)가 커질수록 이전 질문에 대한 대답은 어떻게 바뀔까?

## 5. 이 장에서 나온 것 같은 그래프를 만들 수 있는가? 탐구해 볼만한 다른 것은 무엇인가? 보폭 스케줄러의 그래프는 어떻게 될까?
작업량을 아주많이 추첨권을 아주많이 둘이서 가진다면 처음엔 낮았던 불공정 지표가 점점 1에 가까운근사값으로 올라갈것 같다.
보폭 스케줄러의 그래프는 추첨 스케줄링과 비교했을 때 적은 작업의 길이가 짧거나 반복 횟수가 적을때도 불공정 지표의 값이 보다 더 높을 것(덜 불공정)으로 생각되며
추첨 스케줄링과 마찬가지로 작업의 길이가 길어지거나 반복 횟수가 많아질수록 그래프가 완만해질 것으로 생각된다.
